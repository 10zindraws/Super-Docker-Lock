# Super-Docker-Lock
<img width="57" height="57" alt="2026-01-08 23-01-43" src="https://github.com/user-attachments/assets/2371eb92-28e5-4917-ab96-0eba3040053b" /> <br>
A Krita Python Plugin that allows the user to lock all of their dockers from being closed or resized.
### Features
- Adds a toolbar button that lets you toggle on and off super docker lock
- Hides all dockers title bars for cleaner UI
- Remembers if you left super docker lock on (stays enabled on Krita startup)
